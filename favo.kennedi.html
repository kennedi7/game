elden ring is mij favorieten game omdat er zoveel te doen is
in deze game speel je als een man die alle monsters moest verslaan en zij meer dan honderd verschillende monsters
elke monster heeft verschillende atack paterns die je uit je hoofd moet leren door meer deren keeren dood te gaan
als je tegen een van de bosses vecht dan kan je ook summons gebruiken die kunnen jje helpen vechten tegen de bos
hoe veder je in de game komt hoe meer soorten wapens en armor krijgt elk zwaard of armor geeft je vershilende dingen
je hoeft niet alleen zwaarden te gebruiken je kan oo een van de meerderen magie stafen gebruien om de bosses te ver slaan

archerhero is mijn favorieten mobile game in deze game moet je met een popetje door verschilende dungens gaan
elke keer dat je door een dungeon gaat krijg je geld en levels je kan met je geld kisten open maken
er zijn veel verschillende kisten je je hebt kisten waarvan je armor en wapens krijgt en een waar je matriaalen van krijgr
je hebt ook een die je een darken hoofd geeft die geeft je dan specialen krachten er zijn oo veel verschillende gam modus
ik vind deze game leuk omdat het simpel is en je kan het spelen op de bus terwijl je naar huis gaat
